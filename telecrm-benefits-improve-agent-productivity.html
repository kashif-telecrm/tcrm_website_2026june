Before you request a demo

Questions we always get.

Does the 1-click dialler work on mobile, or only on desktop?
Both. From desktop, the rep presses call inside the lead and the call connects through their phone via telecrm's call sync. On mobile, pressing call dials directly from the device. Call duration, direction and outcome are logged automatically in both cases — the rep doesn't need to enter anything manually after the call.
Does the incoming call pop-up work even when telecrm isn't open on the phone?
Yes. The telecrm mobile app runs a background service that detects incoming calls and surfaces the pop-up even when the app isn't open. The moment a known lead's number calls, the pop-up appears over the native phone dialer — showing their name, current stage, last interaction and a direct link to their lead profile.
Can reps save their own filter views, or is it only a manager-level setting?
Reps can create and save their own filtered views — "My fresh leads from this week," "Callbacks due today," and so on. Saved views load instantly and persist between sessions, so a rep's priority list is waiting for them every morning without any setup. Managers can also create shared views visible to the whole team.
What exactly does Bulk WACA Message let you do?
After filtering a set of leads, a rep can select all and send the same WhatsApp message to the entire filtered set in one action. This uses telecrm's WACA (WhatsApp Cloud API) integration — not the rep's personal WhatsApp — so messages are sent at scale without manual effort. Useful for follow-up nudges, event reminders or re-engagement after a bulk calling session.
What happens when a lead calls from a number that's not saved in telecrm?
If the number isn't already stored against a lead in telecrm, the call is flagged as unknown. telecrm can automatically create a new lead from the incoming number — with the call logged against it — so the rep can update the details immediately after the call ends. Leads with alternate phone numbers stored will also match on the second number.
